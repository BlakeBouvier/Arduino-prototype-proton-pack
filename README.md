# Arduino-prototype-proton-pack
This is code that I started making with Arduino circuits to use in a modded Spirit Halloween proton pack. Using just two Arduino Nano boards and a DFPlayer mini mp3 module.

This code can be run on most arduino boards, a dfplayer mini however, is required for the sound to operate. After which you can use whatever sounds you like, I just got sound files from a range of Ghostbusters media. This code is fairly simple to modify and you can also use whatever colors of LED's you want as well. 

The sfx file is for the dedicated sound board which acts as the master to the seperate light effects board, which recieves comms from the sound board via its serial rx pin. Connected to digital pins 5 and 6 on the sound board a flip switch and momentary switch respectively are highly reccomended.
